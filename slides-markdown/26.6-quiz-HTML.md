# HTML quiz!


## Q1: Which tag is used to create a link to another page?

1. `<p>`
2. `<link>`
3. `<href>`
4. `<a>`
5. `<web>`

### Answer: 4.`<a>`


## Q2: Which element nests directly within the `<ul>` and `<ol>` elements?

1. `<p>`
2. `<ul>`
3. `<aside>`
4. `<h4>`
5. `<li>`

### Answer: 5. `<li>`
