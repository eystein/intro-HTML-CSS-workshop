#  CSS declarations

```
property: value;
```

* Property + Value pairs.
* Divided by a colon `:`
* Finishes with a semi-colon `;`

## Each property has the option of many different values:

```css
color: black;  
color: #24292e;
color: rgb(36, 41, 46);
color: rgba(36, 41, 46, 0.8);
color: hsl(210, 12, 16);
color: hsla(210, 12, 16, 0.8);
```

## _Most_ values can be used with different properties:

```css
    font-size: 50px;
        width: 50px;
margin-bottom: 50px;
      padding: 50px;
        right: 50px;
```
