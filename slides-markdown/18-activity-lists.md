# Create (nested) lists

The list item (`li`) is the only child that's allowed inside a `ul` or `ol`.  

**But** - anything can go inside a `li`!

```HTML
<ul>
  <li>List item 1</li>
  <p>NOPE!</p>
  <li>
    <p>OK, that's better! ðŸ™‚</p>
  </li>
  <li>List item 3</li>
  <li>List item 4</li>
  <li>List item 5</li>
</ul>
```
