# Lists

![List examples](images/list-examples.png)

## Unordered list `<ul>` (bullets)

```html
<ul>
  <li>List item</li>
  <li>List item</li>
  <li>List item</li>
</ul>
```

* List item
* List item
* List item

## Ordered list `<ol>` (numbers)

```html
<ol>
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ol>
```

1. List item 1
2. List item 2
3. List item 3
